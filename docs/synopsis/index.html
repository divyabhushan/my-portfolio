Synopsis

I focus on building continuous integration and deployment process to design, develop (VS Code), version control (Git), re-use (React components), proof-read, deploy (GitHub), and deliver (publish) the docs using Docusaurus static site generator.

A modular code and a CI/CD process in place, helps me edit docs for new feature/bugs/enhancements efficiently.
